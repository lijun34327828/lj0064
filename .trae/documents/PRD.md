## 1. 产品概述
资产收支 AI 测算系统是一款面向高净值人群的个人资产管理可视化工具。用户通过勾选名下资产类目，系统自动测算每月固定收支与现金流状况，并提供保守、稳健、扩张三套资产分配方案，以图表形式直观对比展示。

- 核心目标：帮助用户全面了解资产状况，智能优化资产配置
- 目标用户：拥有多元资产的个人投资者、高净值人群
- 核心价值：提供直观的资产可视化和科学的配置方案建议

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 配置资产、查看测算结果、调整方案比例 |

### 2.2 功能模块
1. **资产配置面板**：资产类目勾选、资产参数设置
2. **收支测算概览**：每月固定收支、现金流盈余/缺口展示
3. **方案对比中心**：保守/稳健/扩张三套方案对比
4. **图表可视化**：环形图展示资产分配、柱状图对比年度收益
5. **比例调节器**：手动调整各类资产投入比例，实时刷新结果

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页（测算台） | 资产类目选择区 | 可勾选地产、基金、私人企业、藏品等资产类目 |
| 主页（测算台） | 参数配置区 | 设置各资产的估值、收益率、月收支等参数 |
| 主页（测算台） | 收支概览卡片 | 展示月固定收入、月固定支出、现金流盈余/缺口 |
| 主页（测算台） | 三套方案对比区 | 展示保守、稳健、扩张三种方案的核心指标 |
| 主页（测算台） | 资产分配环形图 | 以环形图展示各方案的资产配置比例 |
| 主页（测算台） | 年度收益柱状图 | 柱状图对比三套方案的预期年度收益 |
| 主页（测算台） | 比例调节滑块 | 可手动调节各类资产比例，实时刷新测算结果 |

## 3. 核心流程

用户进入页面 → 勾选名下资产类目 → 配置各项资产参数（估值、收益率等） → AI 自动测算收支与现金流 → 生成三套资产分配方案 → 查看图表对比 → 手动调节比例 → 实时刷新结果

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深邃藏青 `#0F172A` 作为背景主色，搭配金色 `#D4AF37` 作为强调色，体现金融专业感与尊贵感
- **辅助色**：翡翠绿 `#10B981` 表示收益/盈余，赤焰红 `#EF4444` 表示支出/缺口
- **按钮风格**：圆角矩形按钮，微渐变背景，hover 时有光泽流动效果
- **字体**：使用 Playfair Display 作为展示字体（标题），Inter 作为正文字体
- **布局风格**：卡片式布局，模块分明，深色主题配合玻璃拟态（Glassmorphism）效果
- **图标风格**：线性图标，金色描边，与整体金融高端风格一致

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主页（测算台） | 顶部标题区 | 大标题、副标题、装饰性分割线 |
| 主页（测算台） | 资产类目选择 | 复选卡片区，选中时有金色边框光晕 |
| 主页（测算台） | 参数配置 | 数字输入框，带单位后缀，渐变焦点效果 |
| 主页（测算台） | 收支概览卡片 | 三张统计卡片，带图标、数值、趋势箭头 |
| 主页（测算台） | 方案选择标签 | 三栏切换标签，选中态有金色下划线 |
| 主页（测算台） | 环形图区域 | 三个并排环形图，悬停显示详情tooltip |
| 主页（测算台） | 柱状图区域 | 三组对比柱状图，渐变填充色 |
| 主页（测算台） | 比例调节区 | 滑块组件，实时数值显示，金色进度条 |

### 4.3 响应式
- 桌面端优先设计（1440px+）
- 中等屏幕（1024-1440px）：图表区域两栏布局调整
- 平板（768-1024px）：卡片堆叠，图表纵向排列
- 移动端（<768px）：单列布局，图表全屏展示

### 4.4 动效设计
- 页面加载：模块渐入上浮动画（stagger 延迟 100ms）
- 卡片 hover：轻微上浮 + 阴影加深 + 金色边框微光
- 数值变化：数字滚动动画（count-up 效果）
- 图表加载：环形图环形描绘动画，柱状图从底部生长动画
- 滑块调节：实时联动图表平滑过渡动画
